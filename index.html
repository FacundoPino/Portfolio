Gestion de Articulos

Puede administrar articulos: agregar, modificar, eliminar, listar, detallar, etc. Use TSQL, C# y .NET, en visual studio y azure.

Carrito de compras

Es un proyecto web. Carga una lista de productos en el home, muestra el detalle de los productos y se agrega al carrito. Una vez agregado se puede manipular la cantidad de el producto(Aumentar, disminuir o eliminar) y pagar. Use C#, ASP.NET, CSS y TSQL.

Gestion de Restaurant

Administra un restaurant desde adentro. Abre y cierra mesas, Carga y cierra pedidos y descuenta stock. Administra (Modifica, agrega y elimina) los insumos, también administra usuarios (agregar, modificar, inactivar y activar). Puede ver reportes sobre sus meseros y mesas filtrados por año, mes y el dia. Cuenta con un login. Una vez ingreso cuenta con permisos que lo separa segun el que ingrese(Puede ser empleado o gerente). Use C#, ASP.Net, TSQL, HTML, CSS y JavaScript

Gestion de Banco

Aplicación FullStack con arquitectura en capas para un banco desarrollada en Java 8, MySQL, CSS, HTML y JavaScript. Trabajando en equipo con GIT y GitHub. El enfoque de esta aplicación es que pueda cumplir con todas o la mayoría de las tareas que requiere un banco como administrador de clientes, cuentas, usuarios, préstamos, transferencias, etc. La aplicación empieza por un login. Este usuario se divide en 2 roles: -Cliente: Tendrá una o varias cuentas para acceder y poder transferir a otras uentas a través del CBU, podrá seleccionar una cuenta y ver su historial de movimientos. Podrá hacer un pedido de préstamo y ademas tendrá un menú de pago de préstamos en el cual visualizará las cuotas que debe pagar y por último tendrá una sección para ver su información personal. -Administrador: ABML de clientes y cuentas, ABM de usuarios, un usuario se dará de alta al crearse un cliente. El administrador maneja la autorización de préstamos mencionada anteriormente. Además visualizará los informes que muestran los movimientos, cuentas, fechas, montos, etc.
